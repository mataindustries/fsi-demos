# FSI Newsletter Demo

Static GitHub Pages demo for a proposed redesign of the Foundation for the Study of Individualism newsletter contents page.

This is a draft review demo only. It does not imply changes to the live website.

## Files

- `index.html` - Demo page content and structure
- `styles.css` - Responsive styling

## How to View

Open `index.html` directly in a browser, or publish this folder with GitHub Pages.

No frameworks, dependencies, package manager, or build step are required.
